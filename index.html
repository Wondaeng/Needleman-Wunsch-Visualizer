Needleman-Wunsch Visualizer

This study-oriented visualizer was created to make the Needleman-Wunsch algorithm easier to understand, and it is based on and references drdrsh/Needleman-Wunsch. Any input change updates the scoring matrix, backtracking path, and final alignment in real time.

Backtracking Path

Tie-break rule for one selected traceback path: Diagonal → Up → Left. If multiple directions have the same max score, all are still shown as best-direction arrows in the matrix.

Hover or click a matrix cell to highlight diagonal/up/left origin cells and see candidate calculations on the right.
