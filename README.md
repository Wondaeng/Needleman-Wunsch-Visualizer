# Needleman-Wunsch Visualizer

Interactive web visualizer for the Needleman-Wunsch algorithm:  
https://needleman-wunsch-visualizer.vercel.app/

This repository is based on [drdrsh’s Needleman-Wunsch repository](https://github.com/drdrsh/Needleman-Wunsch).

The main features added in this version are:
- displaying the intermediate alignments for each cell for all possible cases (diagonal, top, and left)
- visual enhancements to improve readability
